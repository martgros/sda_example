# SDA Example

Small reference project for sequential decision analytics examples in Python.

## What Is Here

- `sda_example_simple`: stable, lightweight example for getting started.
- `sda_example`: work in progress; not yet documented for general use.
- `notebooks/`: interactive demos and policy-analysis notebooks.

## Quick Start

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -e .
pytest
```

## Start With

- [Simple example README](src/sda_example_simple/README.md)
- [Simple example notebook](notebooks/sda_example_simple_demo.ipynb)

The main `sda_example` package is currently WIP. Use `sda_example_simple` if you want a clear entry point.
